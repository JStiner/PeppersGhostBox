
# Pepper's Ghost Hologram Display (DIY Build)

## Overview
This project documents the design and construction of a Pepper’s Ghost illusion display built using a wooden frame, acrylic panel, and digital projector. The goal is to create a convincing floating “hologram” effect using relatively inexpensive materials and careful control of geometry, lighting, and contrast.

The build documented here uses:

- 18 × 24 inch acrylic panel
- Wooden frame constructed from 2×4 lumber
- 45° acrylic angle for reflection
- Rear-positioned projector
- Matte gray projection surface
- Black interior light control (paint and blackout curtains)

This repository captures the final working configuration and lessons learned.

---

# How Pepper’s Ghost Works

The illusion works by reflecting a projected image off a piece of transparent acrylic positioned at 45 degrees.

Light path:

Projector → projection surface → reflection in acrylic → viewer

# Materials

## Structural
- 2×4 lumber
- wood screws
- wood glue

## Optical
- 18×24 acrylic sheet
- two‑way mirror film
- projector

## Light control
- flat black paint
- blackout curtain fabric
- projector hood

## Projection surface options
Best → projection screen material

Good → matte gray fabric or photography backdrop

Acceptable → matte white board

Avoid → glossy surfaces

---

# Lessons Learned

1. Projector alignment is critical
2. Interior light control matters more than brightness
3. Acrylic thickness affects reflection quality
4. Two‑way mirror film improves brightness
5. Screen placement affects reflection efficiency
6. Hide projection surface edges
